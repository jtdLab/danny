# {{name}}_cli

A file-based CLI developed using [danny_cli](https://pub.dev/packages/danny_cli).

## Quick Start

```sh
# 🎯 Activate from https://pub.dev
dart pub global activate danny_cli

# 📦 Build the project
danny build

# 🔨 Activate the project locally
danny activate

# Use the CLI
{{name}} -h
```
